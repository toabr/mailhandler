### Features:

* Node.js Backend on port 4000
*  Express
*  mysql 

* React Frontend on port 3000
* react-router
* node-sass
* Bootstrap 4
* font-awesome

### Installation

```
$npm i && cd frontend && npm i
```

* change mysql- db, user and password
* import emails.sql table

### Start Development Server

```
cd root and $npm run dev
```
